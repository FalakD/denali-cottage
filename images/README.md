# Property Photos

To use your own photos from the Zillow listing:

1. Visit the [Zillow listing](https://www.zillow.com/homedetails/1767-26th-Ave-E-Seattle-WA-98112/450900007_zpid/) and save the images you want to use.
2. You can use a tool like [Zillow Photos Downloader](https://github.com/bacon1933/ZillowPhotosDownloader) to bulk download, or right-click and save each photo.
3. Add your images to this folder with descriptive names (e.g., `exterior.jpg`, `living-room.jpg`, `kitchen.jpg`).
4. Update the image paths in `index.html` to reference your local files (e.g., `images/exterior.jpg` instead of the Unsplash URLs).

Current placeholder images are from Unsplash and depict similar Seattle-style homes.
